# 📖 Story Scripts Website

## 📌 Project Overview
The **Story Scripts** project is a creative web-based platform designed to present **short stories, scripts, or written content** in an organized and visually appealing format.

This website allows users to read and explore different stories in a simple and engaging way, making it useful for both **entertainment and learning purposes**.

**Access project here:** https://kowshika555.github.io/story_scripts/

---

## 🚀 Features
- 📚 Collection of story scripts  
- 🖥️ Clean and user-friendly interface  
- 📱 Responsive design (mobile & desktop)  
- 🎨 Simple and readable layout  
- ⚡ Fast loading static website  
- 🔗 Easy navigation between stories  

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout design  
- **JavaScript** – Interactivity (if applied)

## 💡 How It Works
1. Open the website  
2. Browse the available story scripts  
3. Click on a story to read  
4. Navigate between different scripts easily
